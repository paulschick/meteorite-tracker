Project NEO

Angular Demonstration

This is a project used to learn Angular through concept application and relentless refactoring. The name NEO comes from Nasa's term, "Near Earth Objects," used to describe asteroids and comets in close proximity to the Earth.

What started off as a sandbox for writing API requests to learn RxJS Operators has evolved into a full-fledged application. The future of this project is to better visualize the open data from the NASA RESTful API.
